Для того, чтобы нанести на изображения (формата jpg или JPG) водный знак и логотип, необходимо сделать следующее:

1) Поместить изображения в папку input;
2) Запустить файл water_mark.exe;
3) Дождаться завершения работы запущенного сервиса;
4) Забрать изображения с нанесенными водным знаком и логотипом из папки output.

При старте сервис открывает консольное окно, в которое выводит краткий отчет о выполняющейся работе.
После завершения работы сервиса консольное окно с отчетом остается открытым (его можно закрыть вручную).

Если при попытке сервиса сохранить файл в папку output в ней уже есть файл с таким же именем, то существующий файл перезаписывается.
